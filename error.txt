1. SyntaxError — помилка, яка виникає якщо інтерпретатор виявляє помилку синтаксису в коді. Простими словами, Python не розуміє яку дію від нього очікують. 
Приклад: 
[print(Hello World)]

2. NameError — помилка, яка виникає якщо спробувати використати змінну, функцію або модуль які не були оголошені. Простими словами, Python дуже уважно слухав батьків і не веде справи з незнайомцями :)
Приклад: 
[a = 5 
b = 10
print(x + y)]

3. TypeError — помилка, яка виникає, коли тип даних об’єкта в операції є невідповідним. Простими словами, не можна число 2 поділити на слово «їжак»
Приклад:
[a = 'John'
b - 23
print(name + age)]

AttributeError — помилка, яка виникає, якщо посилання на метод для певного об’єкта не існує. Тобто в переліку всіх доступних дій над об’єктом Python не знаходить потрібний. Простими словами, птахи можуть літати, а їжаки ні
Приклад:
[a = 25
a.append(1)]

ValueError — помилка, яка виникає, коли функція отримує аргумент правильного типу, але невідповідного значення. Простими словами, якщо називати їжака орлом і посадити його висиджувати яйця, він все одно не стане птахом
Приклад:
[a = int('text')
print(a)]

ZeroDivisionError — помилка, яка виникає при спробі поділити число на 0. 
Приклад:
[a = 10
b = 0
print(a/b)
]
